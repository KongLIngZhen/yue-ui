# yue-ui

> Vue and Bootstrap Component Library.

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build yue-ui.min.js for production with minification
npm run build:all

# build lib for production with minification
npm run build:lib

# build css for production with minification
npm run build:style

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test
```
